    Help

    Welcome to the settings app. This app will allow you to set the options for your simulator without having to change the config.xml file directly. You can also use this app to enable, disable, and change settings for any apps running in this simulator. You can also disable this app itself. A password can also be set to block access by non-authorized personnel.
